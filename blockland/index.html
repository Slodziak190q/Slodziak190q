blockland code

blockland is a lego-building game by Eric Hartman. it's kinda dead now, because Brickadia...exists.

with that being said, my projects have also died. i'm not working on them anymore in favour of web design. every single old addon i've made for the game is available here, with the github repositories to download them.

to install them, click the link and download the zip file. do not unzip it. drag it into your add-ons folder (in the Steam folder, it's usually located in steamapps/common/Blockland/Add-Ons) and restart blockland.

Client_RapidClickFire

Repository

allows players to fire their weapons rapidly by holding a certain keybind, or click left or right click extremely fast.

Script_PingUtils

Repository / Blockland Glass

check a certain player's latency to the server. includes a special function to change a user's score based on their ping.

Server_PokePM

Repository

send users private whispers in chat, and poke them to come back from being afk.

Server_StackPlayers

Repository

stack onto other users. or stack others onto you. or stack others onto others.

Server_StatusModv2

Repository

go afk, go be right back, auto afk. old addon updated for modern servers. even changes the player's shapename!

Server_YAMA

Repository / Blockland Glass

yet another moderator addon. most notable addon and ambitious project i've made.

Support_CustomModuleLoader

Repository

client-sided module loader. useful for small scripts you don't want in their own addon.

System_NewGameBar

Repository

updated game bar from rtb (return to blockland).
